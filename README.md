# CNN-based model for bone cancer detection from X-Ray images.

This project implements a Convolutional Neural Network (CNN) in PyTorch to classify bone X-Ray images as cancerous or normal. The model is trained and evaluated on a dataset of labeled X-Ray images.

## Dataset

The dataset used for this project is from [Kaggle: Bone Cancer Detection Dataset](https://www.kaggle.com/datasets/ziya07/bone-cancer-detection-dataset).  
It contains X-Ray images labeled as either cancer or normal.
